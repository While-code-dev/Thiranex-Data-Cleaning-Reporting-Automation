# Data Cleaning & Reporting Automation

## Overview

This project is a Streamlit-based web application that automates the process of data cleaning and reporting. Users can upload CSV files, clean missing and duplicate data, generate summary reports, visualize insights, and download the cleaned dataset.

## Features

- Upload CSV datasets
- Automatic removal of duplicate records
- Automatic handling of missing values
- Data quality improvement using Python and Pandas
- Automated reporting and summary statistics
- Interactive visualizations
- Download cleaned dataset
- User-friendly Streamlit dashboard

## Technologies Used

- Python
- Pandas
- Streamlit
- Matplotlib

## How It Works

1. Upload a CSV file.
2. The system automatically:
   - Removes duplicate records
   - Fills missing values
   - Generates summary metrics
3. Visual reports and charts are displayed.
4. Download the cleaned dataset with a single click.
